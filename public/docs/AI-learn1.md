## 二、攻克第3天：RAG技术实战（全免费工具）

RAG的核心是“让AI能读你的专属文档，回答针对性问题”，比如用公司产品手册训练AI，自动回复客户咨询。咱们用免费工具搭一个“职场文档问答助手”，学完就能用在工作中。

### 1. 免费工具准备（3分钟搞定）

不用复杂部署，选2个新手友好的工具：

- **本地模型（替代付费API）**：用Ollama（免费，支持Windows/Mac），能跑Llama 3（8B参数）等开源模型，足够日常用
- **RAG框架（替代付费向量数据库）**：用LangChain（免费Python库）+ 本地文件（Excel/Word），不用花钱买存储

### 2. 实战步骤：搭建“职场文档问答助手”

以“用公司产品手册训练AI，自动回答客户问题”为例，分3步走：

#### 步骤1：准备“专属知识库”（5分钟）

- 把公司产品手册（比如“儿童保温杯说明书”）转成Excel，列2栏：`问题|答案`（比如“保温杯保温时长？| 6小时保热，12小时保冷”）
- 若无手册，直接在Excel里写10-20个常见问题+答案（比如客户常问的“售后政策”“尺寸选择”）

#### 步骤2：安装工具并运行基础代码（10分钟）

1. 安装Ollama：
   - 官网下载Ollama（https://ollama.com/），安装后打开命令行，输入 `ollama pull llama3`，等待模型下载完成（约10分钟，视网速而定）
2. 安装LangChain：  
   打开命令行，输入 `pip install langchain openpyxl`（需要先装Python，官网免费下载）
3. 复制以下代码，保存为`rag_assistant.py`（把代码里的“产品手册.xlsx”换成你的文件路径）：

   ```python
   from langchain.document_loaders import OpenpyxlLoader
   from langchain.embeddings import OllamaEmbeddings
   from langchain.vectorstores import FAISS
   from langchain.chains import RetrievalQA
   from langchain.llms import Ollama

   # 1. 加载你的产品手册（知识库）
   loader = OpenpyxlLoader("产品手册.xlsx")
   docs = loader.load()

   # 2. 把文档转成“向量”（让AI能检索）
   embeddings = OllamaEmbeddings(model="llama3")
   db = FAISS.from_documents(docs, embeddings)

   # 3. 搭建问答链
   llm = Ollama(model="llama3")
   qa_chain = RetrievalQA.from_chain_type(
       llm=llm,
       chain_type="stuff",
       retriever=db.as_retriever()
   )

   # 4. 测试：问一个客户常问的问题
   question = "保温杯可以装碳酸饮料吗？"
   result = qa_chain.run(question)
   print("AI回答：", result)
   ```

4. 运行代码：命令行输入 `python rag_assistant.py`，就能看到AI根据你的手册回答问题
